## Quarterly Retest Protocol (end of every 12-week phase)

1. Full bloodwork panel — same fasted conditions, same lab
2. DEXA scan — same facility for consistent calibration
3. Longevity benchmarks — same day, same rested state
4. Progress photos — same lighting, same angles
5. Update [HEALTH_BRAIN.md](../HEALTH_BRAIN.md) dashboard with new values
6. Update [01_bloodwork.md](01_bloodwork.md) trend table
7. Log protocol changes and outcomes in [16_protocol_changes.md](16_protocol_changes.md)
